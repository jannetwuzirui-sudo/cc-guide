# 实战：博客站

> 接单价格区间：1000-3000 元 | 开发周期：2-5 天 | 难度：⭐⭐⭐

博客站是中等难度的接单项目，客户通常是个人博主、自媒体人或小型团队。相比静态官网，博客站需要考虑内容管理、SEO 和长期维护。

---

## 一、需求分析

### 典型客户需求

客户是一位科技领域的自媒体博主，需要一个个人博客：

- 首页文章列表（带封面图、摘要、标签）
- 文章详情页（Markdown 渲染、目录导航、代码高亮）
- 分类和标签归档页
- 关于页面
- 搜索功能
- RSS 订阅
- 暗色模式切换
- 评论系统集成

### 框架选择

| 框架 | 优势 | 适用场景 |
|------|------|---------|
| Astro | 性能极佳，生态丰富 | 内容为主的博客 |
| Hugo | 构建速度最快 | 大量文章的博客 |
| Next.js | 功能灵活 | 需要动态功能的博客 |
| VitePress | 配置简单 | 技术文档型博客 |

本案例选用 Astro，因为它对内容站点的支持最好，且部署简单。

---

## 二、Claude Code 提示词

### 第一步：项目搭建

```
用 Astro 创建一个博客项目，要求：
1. 使用 Astro 的 Content Collections 管理文章
2. 文章用 Markdown 编写，支持 frontmatter（title, date, tags, cover, summary）
3. 首页显示最新 10 篇文章，分页展示
4. 侧边栏显示标签云和最近文章
5. 使用 TailwindCSS 做样式
6. 支持暗色模式（跟随系统 + 手动切换）
```

### 第二步：文章功能完善

```
完善博客的文章详情页：
1. 文章顶部显示标题、日期、标签、预计阅读时间
2. 右侧浮动目录导航（根据 h2/h3 自动生成）
3. 代码块支持语法高亮和一键复制
4. 文章底部显示上一篇/下一篇导航
5. 相关文章推荐（基于相同标签）
6. 图片点击放大查看
```

### 第三步：SEO 优化

```
为博客添加 SEO 优化：
1. 每个页面的 meta title 和 description
2. Open Graph 标签（分享到社交媒体时的预览）
3. 自动生成 sitemap.xml
4. 自动生成 RSS feed
5. 结构化数据标记（Article schema）
6. robots.txt 配置
```

### 第四步：搜索和评论

```
为博客添加以下功能：
1. 客户端搜索（使用 Pagefind 或 Fuse.js）
2. 集成 Giscus 评论系统（基于 GitHub Discussions）
3. 文章阅读进度条
4. 返回顶部按钮
```

### 内容管理方案

对于不懂技术的客户，提供以下方案：

1. **直接编辑 Markdown** — 教客户用 Typora 等编辑器
2. **搭配 Decap CMS** — 提供可视化后台，通过 Git 管理内容
3. **Notion 同步** — 客户在 Notion 写作，自动同步到博客

### 主题定制要点

- 字体选择：中文推荐思源黑体或霞鹜文楷
- 文章排版：行高 1.8，段落间距适当
- 代码块：选择护眼的配色方案
- 图片：统一圆角和阴影风格

## 四、部署上线

### 推荐部署方案

```
帮我配置 Astro 博客的自动部署：
1. 代码推送到 GitHub
2. Vercel 自动构建部署
3. 绑定自定义域名
4. 配置 SSL 证书
```

### 部署平台对比

| 平台 | 费用 | 国内速度 | 自动部署 |
|------|------|---------|---------|
| Vercel | 免费 | 一般 | 支持 |
| Netlify | 免费 | 一般 | 支持 |
| Cloudflare Pages | 免费 | 较快 | 支持 |
| 阿里云 OSS | 约 10 元/月 | 快 | 需配置 |

如果客户主要面向国内读者，建议用 Cloudflare Pages 或阿里云方案。

| 项目内容 | 参考价格 |
|---------|---------|
| 基础博客（主题 + 部署） | 1000-1500 元 |
| 含搜索和评论系统 | 1500-2000 元 |

### 维护建议

博客是长期项目，建议和客户签订维护协议：

- 月度维护（更新依赖、备份）：100-200 元/月
- 功能迭代按需报价
- 紧急故障修复：响应时间和费用提前约定
