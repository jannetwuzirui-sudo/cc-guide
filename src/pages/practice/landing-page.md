# 实战：营销落地页

> 接单价格区间：300-1000 元 | 开发周期：0.5-2 天 | 难度：⭐⭐

营销落地页（Landing Page）是单页面设计，核心目标只有一个：转化。无论是收集用户信息、引导下载还是促成购买，每个元素都要为转化服务。

---

## 一、需求分析

### 典型客户需求

客户是一家在线教育机构，需要一个课程推广落地页：

- 课程核心卖点展示（Hero 区域）
- 课程特色/亮点（3-6 个 Feature 卡片）
- 学员评价/成功案例
- 讲师介绍
- 价格方案对比
- 报名表单 / CTA 按钮
- 倒计时促销模块

### 转化要素清单

一个高转化落地页必须包含：

1. **吸引力标题** — 3 秒内抓住注意力
2. **社会证明** — 学员数量、好评截图、合作品牌
3. **痛点共鸣** — 描述目标用户的困境
4. **解决方案** — 你的产品如何解决问题
5. **行动号召** — 醒目的 CTA 按钮，全页多次出现

## 二、Claude Code 提示词

### 第一步：页面框架

```
创建一个高转化营销落地页，单页面 HTML + TailwindCSS，包含以下模块（从上到下）：
1. 顶部导航（固定，含 Logo 和 CTA 按钮）
2. Hero 区域（大标题 + 副标题 + CTA 按钮 + 背景图）
3. 数据展示条（学员数、好评率、课程数 三个数字动画）
4. 课程特色（3列卡片，图标 + 标题 + 描述）
5. 学员评价（轮播卡片，头像 + 姓名 + 评价内容）
6. 价格方案（3列对比表，中间推荐方案高亮）
7. 常见问题 FAQ（手风琴折叠）
8. 底部 CTA（大按钮 + 倒计时）
9. Footer

主色调 #ff6b35，背景白色，中文排版。
```

### 第二步：交互增强

```
为落地页添加以下交互效果：
1. 滚动时各模块淡入动画（Intersection Observer）
2. 数字统计区域的数字递增动画
3. 倒计时组件（天:时:分:秒）
4. CTA 按钮 hover 时的脉冲动画
5. 点击 CTA 弹出报名表单弹窗
6. 表单提交后显示成功提示
```

### 第三步：移动端优化

```
优化落地页的移动端体验：
1. Hero 区域文字缩小，按钮全宽
2. 价格方案改为横向滑动卡片
3. 底部固定一个悬浮 CTA 按钮栏
4. 图片懒加载
5. 确保首屏加载速度在 3 秒内
```

```
└── thank-you.html
```

### 转化率优化技巧

- CTA 按钮颜色要和页面主色形成对比
- 每屏至少出现一个 CTA 入口
- 表单字段越少越好（姓名 + 手机号足矣）
- 添加紧迫感元素（限时优惠、名额有限）
- 页面加载速度直接影响转化率

### 响应式设计要点

```css
/* 关键断点 */
@media (max-width: 768px) {
  .hero-title { font-size: 1.75rem; }
  .price-grid { display: flex; overflow-x: auto; }
  .floating-cta { display: fixed; bottom: 0; }
}
```

## 四、部署上线

### 推荐方案

落地页通常需要快速上线，推荐：

1. **Vercel**（免费）— 推送即部署，自带 CDN
2. **阿里云 OSS**（低成本）— 国内访问快，适合投放广告
3. **腾讯云静态托管**（低成本）— 微信生态内访问友好

### A/B 测试考虑

如果客户有投放预算，建议准备两个版本：

- 版本 A：强调价格优惠
- 版本 B：强调学习效果

用不同的 URL 参数区分，通过统计工具对比转化数据。

### 数据追踪

```
帮我在落地页中集成以下追踪代码：
1. 百度统计 / Google Analytics
2. CTA 按钮点击事件追踪
3. 表单提交成功事件追踪
4. 页面滚动深度追踪（25%/50%/75%/100%）
```

## 小结

落地页项目虽然页面少，但对设计感和转化思维要求较高。用 Claude Code 可以快速生成页面结构和交互代码，你需要重点关注的是：

1. 理解客户的转化目标，所有设计围绕目标展开
2. 移动端体验是重中之重（大部分流量来自手机）
3. 积累高转化模板，复用提高效率
4. 交付后跟进数据表现，用结果说话争取复购
