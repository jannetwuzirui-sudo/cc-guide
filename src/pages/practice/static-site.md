# 实战：静态企业官网

> 接单价格区间：500-2000 元 | 开发周期：1-3 天 | 难度：⭐⭐

企业官网是淘宝/闲鱼上最常见的外包订单类型。客户通常是小微企业主，需要一个简洁专业的线上门面。本篇将完整演示如何用 Claude Code 高效交付一个静态企业官网。

---

## 一、需求分析

### 典型客户需求

客户是一家做智能家居安装的小公司，提出以下要求：

- 公司简介页面（关于我们、团队介绍）
- 产品/服务展示（6-8 个产品卡片，带图片和描述）
- 新闻动态（静态文章列表）
- 联系我们（地址、电话、地图嵌入、留言表单）
- 手机端适配
- 需要有微信二维码展示

### 需求确认清单

在开工前，务必和客户确认：

1. 是否有品牌 VI（Logo、主色调、字体）
2. 文案和图片素材谁提供
3. 是否需要后台管理
4. 域名和服务器谁负责
5. 后续维护怎么算

## 二、Claude Code 提示词

### 第一步：项目初始化

```
帮我创建一个静态企业官网项目，使用 HTML + TailwindCSS（CDN引入），
包含以下页面：
- index.html（首页，含导航、Banner、公司简介、服务亮点、底部）
- products.html（产品展示页，6个产品卡片网格布局）
- about.html（关于我们，公司历程时间线）
- contact.html（联系我们，含表单和地图占位）

要求：
1. 响应式设计，移动端优先
2. 中文排版，使用系统字体栈
3. 主色调 #1a56db，辅助色 #f3f4f6
4. 所有页面共享统一的 header 和 footer
```

### 第二步：细化各页面

```
优化 products.html：
1. 产品卡片加上 hover 放大效果
2. 每个卡片包含：图片、产品名、简短描述、"了解更多"按钮
3. 加一个筛选标签栏（全部/智能灯光/智能安防/智能家电）
4. 用 JavaScript 实现简单的标签筛选功能
```

### 第三步：移动端适配检查

```
检查所有页面的移动端适配问题：
1. 导航栏在手机上改为汉堡菜单
2. 产品卡片在手机上改为单列
3. 联系表单在手机上全宽显示
4. 检查字体大小是否合适
```

## 三、开发过程

### 项目文件结构

```
corporate-site/
├── index.html
├── products.html
├── about.html
├── contact.html
├── css/
│   └── custom.css
├── js/
│   └── main.js
├── images/
│   ├── logo.png
│   ├── banner.jpg
│   └── products/
└── favicon.ico
```

### 开发顺序建议

1. 先搭好公共部分（header、footer、导航）
2. 完成首页布局和内容填充
3. 逐个完成子页面
4. 统一调整响应式样式
5. 替换客户提供的真实素材
6. 浏览器兼容性测试

### 常见坑点

- 客户给的图片尺寸不统一 → 用 `object-fit: cover` 统一处理
- 中文排版行高不够 → `line-height` 建议设为 1.8
- 微信内置浏览器兼容 → 避免使用太新的 CSS 特性
- 表单提交 → 静态站可用 Formspree 或 腾讯云函数处理

## 四、部署上线

### 推荐部署方案

对于静态企业官网，推荐以下方案（按成本排序）：

1. **GitHub Pages**（免费）→ 适合懂技术的客户
2. **Vercel / Netlify**（免费额度）→ 自动部署，速度快
3. **阿里云 OSS + CDN**（约 10 元/月）→ 国内访问快，适合正式项目
4. **宝塔面板 + 轻量服务器**（约 50 元/月）→ 客户最容易理解

### 用 Claude Code 辅助部署

```
帮我写一个部署脚本，将这个静态网站部署到阿里云 OSS：
1. 压缩所有图片（使用 sharp 或 imagemin）
2. 压缩 HTML/CSS/JS
3. 上传到指定 OSS bucket
4. 配置自定义域名的说明文档
```

### 域名绑定

提醒客户准备好已备案的域名，协助完成：
- DNS 解析配置（A 记录或 CNAME）
- SSL 证书申请（Let's Encrypt 免费证书）
- 强制 HTTPS 跳转

## 五、交付客户

### 交付物清单

- [ ] 全部源代码（压缩包或 Git 仓库）
- [ ] 部署后的线上地址
- [ ] 简单的修改说明文档（告诉客户怎么改文字和图片）
- [ ] 素材源文件备份

### 售后约定

建议在交付时明确：

- 免费修改次数（通常 2-3 次小改）
- 超出范围的修改按次收费（50-100 元/次）
- 服务器/域名续费提醒
- 后续功能新增另行报价
