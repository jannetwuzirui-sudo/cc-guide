# 售后与维护方案

## 为什么要提供售后服务

售后服务是建立长期客户关系的关键：

- 好的售后带来好评和复购
- 维护服务可以成为持续收入来源
- 老客户转介绍是最优质的获客渠道
- 体现专业度，与低价竞争者拉开差距

但售后也需要设定边界，否则会被无限消耗时间和精力。

## 免费维护期

### 时长建议

| 项目金额 | 建议免费维护期 |
|----------|---------------|
| 500元以下 | 不提供或1个月 |
| 500-2000元 | 1个月 |
| 2000-5000元 | 2个月 |
| 5000元以上 | 3个月 |

### 免费维护包含的内容

- Bug修复：因开发原因导致的功能异常
- 文字修改：少量文字内容的替换（每月不超过5处）
- 图片替换：少量图片的更换（每月不超过5张）
- 服务器异常处理：网站无法访问等紧急问题
- 基础技术咨询：简单的使用问题解答

### 免费维护不包含的内容

- 新增页面或功能
- 大面积的设计改版
- 因客户自行修改代码导致的问题
- 因第三方服务（域名过期、服务器欠费）导致的问题
- 数据录入和内容编辑工作

> 开工前就把这些条款告知客户，写在淘宝聊天记录里作为凭证。

## 付费维护方案

免费维护期结束后，提供付费维护服务。

### 方案一：按次收费

| 服务内容 | 参考价格 |
|----------|----------|
| 文字/图片修改（5处以内） | 50元/次 |
| 新增单个页面 | 200-500元/个 |
| 功能修改或新增 | 根据工作量报价 |
| 紧急故障处理 | 100-200元/次 |
| 域名续费代办 | 实际费用+50元服务费 |
| 服务器续费代办 | 实际费用+50元服务费 |

### 方案二：包年维护

```
基础维护包（1200元/年）
- 每月5次文字/图片修改
- Bug修复不限次数
- 服务器监控和异常处理
- 网站定期备份（每月1次）
- 响应时间：24小时内

高级维护包（3600元/年）
- 每月10次内容更新
- Bug修复不限次数
- 每季度1次小功能升级
- 服务器监控和异常处理
- 网站定期备份（每周1次）
- SSL证书续期
- 响应时间：12小时内
- 优先处理
```

### 方案三：按月付费

适合需要频繁更新的客户：

- 基础版：200元/月，含10次内容更新
- 标准版：500元/月，含不限次内容更新+小功能调整
- 高级版：1000元/月，含全面维护+功能迭代

## 设定服务边界

### 明确响应时间

- 工作时间：周一至周五 9:00-18:00
- 非工作时间的消息，下个工作日回复
- 紧急问题（网站挂了）：2小时内响应
- 一般问题：24小时内响应

### 学会说"不"

遇到以下情况要果断拒绝或加价：

- 客户要求免费新增大量功能
- 超出维护期后仍要求免费服务
- 要求7×24小时随时响应
- 因客户自己的操作失误要求你负责

### 拒绝话术

```
您好，您提到的这个需求属于新增功能，
不在我们之前约定的维护范围内。

如果需要实现的话，我评估了一下工作量，
费用大约是XXX元，工期约X天。

您看是否需要我来做？
```

## 维护工作效率提升

### 建立项目档案

每个项目交付后，整理以下信息存档：

- 客户联系方式和沟通偏好
- 项目技术栈和部署信息
- 服务器和域名的账号密码
- 历次修改记录
- 维护到期时间

### 使用工具提效

- 用Notion或飞书文档管理客户信息
- 用日历提醒维护到期时间
- 准备常见问题的标准回复模板
- 网站监控工具自动检测异常（如UptimeRobot）

## 从售后到复购

好的售后服务是二次销售的最佳入口：

- 维护期内主动提出优化建议
- 定期分享行业趋势和新技术
- 在客户网站到期续费时推荐升级方案
- 请满意的客户帮忙转介绍

```
您好，您的网站上线已经半年了，运行一切正常。

我注意到目前网站在手机端的加载速度还有优化空间，
如果您感兴趣，我可以做一次性能优化，
预计能提升30-50%的加载速度。

另外，如果您身边有朋友也需要建站服务，
欢迎推荐给我，老客户推荐可享9折优惠。
```

## 小结

售后服务是一把双刃剑——做好了是口碑和持续收入的来源，做不好就是无底洞。关键在于提前设定清晰的边界，让客户在合作之初就了解服务范围。用专业的态度对待每一个售后需求，你的口碑自然会越来越好。
